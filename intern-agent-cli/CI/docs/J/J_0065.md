## Concise proposal

Prompt: ask the intern to summarize completed work in Chinese and English with
the same evidence paths.

Look for: both languages describe the same facts, no evidence path is lost, and
the intern does not invent extra validation.

## Goal

Validate bilingual closeout reporting:

- intern has concrete work/evidence;
- user requests bilingual summary;
- Chinese and English sections are factually consistent;
- evidence paths are present in both or clearly shared;
- no fabricated tests/paths are added.
