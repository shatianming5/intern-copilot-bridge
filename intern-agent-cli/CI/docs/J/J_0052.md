## Retired reason

Session 12 scope audit retired this case. It tests whether an agent follows a
generic user instruction to avoid tools/edits, not a plugin behavior, product
journey, or explicit built-in prompt contract.

## Concise proposal

Prompt: ask a conceptual question and explicitly say "不要运行命令，不要改文件".

Look for: intern answers from existing context without tool execution, and
reports uncertainty instead of fabricating.

## Goal

Validate no-tools/no-edit instruction following:

- user asks for explanation only;
- intern does not run shell commands or edit files;
- answer is concise and bounded by known context;
- if evidence is insufficient, intern states that limitation.

## Why J

This is a real conversation mode and protects user time/state. It is not a CLI
contract.
