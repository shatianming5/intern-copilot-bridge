# J_0092 Same-name intern session isolation journey

<!-- METADATA:TYPE=J,ID=J_0092,STATUS=ReviewDraft,OWNER=intern_ci_lead,SESSION=9 -->

## Concise proposal

Prompt: talk to two same-name interns in different projects.

Look for: each Feishu group and session replies from the intended project; no
tmux/session cross-talk.

## Goal

Validate same-name session isolation:

- two projects have the same intern name;
- each group sends a unique prompt token;
- replies come from the correct project/session;
- restart or follow-up keeps the project binding.

## Why J

Compound-key/session map checks are F. This case proves the real user can talk
to the intended intern without cross-project leakage.

- Same-name interns do not steal each other's prompts.
