# J_0063 User supplies patch snippet journey

<!-- METADATA:TYPE=J,ID=J_0063,STATUS=Retired,OWNER=intern_ci_lead,SESSION=12 -->

## Retired reason

Session 12 scope audit retired this case. Applying a user-supplied patch snippet
is generic coding agent behavior and should not be a standalone product J
journey.

## Concise proposal

Prompt: provide a small patch snippet and ask the intern to apply it after
checking context.

Look for: intern verifies the target context, applies the patch only if it fits,
and reports mismatch instead of forcing it.

## Goal

Validate user-supplied patch handling:

- user provides patch-like text;
- intern checks file context;
- exact or compatible patch is applied;
- incompatible patch is refused with evidence;
- no unrelated edits occur.

## Why J

This is a natural user collaboration pattern and requires agent judgment.
