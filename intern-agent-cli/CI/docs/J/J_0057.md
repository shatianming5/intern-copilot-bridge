## Scope note

Keep this case only as built-in prompt coverage for the explicit no-web/local
answer policy. Do not treat it as a generic "agent follows any instruction"
test.

## Concise proposal

Prompt: ask a stable repo-local question and explicitly say not to browse or
use external latest information.

Look for: intern answers from local repo context only and does not browse or
claim up-to-date external facts.

## Goal

Validate local-only instruction following:

- user requests a repo-local answer;
- user says no web/latest verification;
- intern uses local files or existing context only;
- reply does not cite fabricated external facts.

## Why J

This is a real conversational control path for avoiding unnecessary external
calls and keeping answers grounded in the workspace.
