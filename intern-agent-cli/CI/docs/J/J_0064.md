# J_0064 User asks to preserve formatting journey

<!-- METADATA:TYPE=J,ID=J_0064,STATUS=Retired,OWNER=intern_ci_lead,SESSION=12 -->

## Retired reason

Session 12 scope audit retired this case. Preserving formatting on a generic
edit is ordinary agent behavior unless tied to an explicit built-in prompt
contract; it is not a product journey by itself.

## Concise proposal

Prompt: ask for a tiny edit in a file with unusual formatting and say preserve
formatting.

Look for: intern changes only the requested semantic content and does not run a
broad formatter.

## Goal

Validate preserve-formatting instruction:

- file has intentional unusual whitespace/order;
- user requests one tiny semantic edit;
- intern preserves surrounding formatting;
- no broad formatting churn occurs.

## Why J

This is a user preference expressed in natural language, not a static formatter
contract.
