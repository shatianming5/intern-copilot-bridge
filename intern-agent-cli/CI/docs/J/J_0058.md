## Scope note

Keep this case only as built-in prompt coverage for current/latest verification
policy. It should verify source/date grounding, not generic answer quality.

## Concise proposal

Prompt: ask a time-sensitive external question and explicitly ask the intern to
verify the latest source.

Look for: intern uses an approved external lookup path, cites source evidence,
and separates current facts from repo-local assumptions.

## Goal

Validate latest-information instruction following:

- user asks for current/latest information;
- intern performs approved verification;
- reply cites source/date evidence;
- repo-local assumptions are separated from current external facts.

## Why J

This is a real user trust journey. It is not a product API contract, but it
affects whether agent answers are grounded.
