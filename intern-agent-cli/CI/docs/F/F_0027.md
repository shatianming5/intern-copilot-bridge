## 通过标准

- Codex session GUI command 与 CLI/session contract 一致。
- prerequisite gate 失败不产生副作用。
- start/restart 不发送业务 prompt，restart 失败保留可诊断现场。
- no-prompt restart 后 session/tmux/status running；不要求 Codex session id。same-session resume 需要真实 agent turn，归 J 用例。

## 失败分类

- GUI 使用非 Codex session path：产品 bug。
- prerequisite 失败后留下 tmux/session map：产品 bug。
- CI 用 mock prompt 代替 session command：CI 逻辑错误。
- restart 后 session/tmux 不 running 或 registry 指向 dead session：产品 bug。
