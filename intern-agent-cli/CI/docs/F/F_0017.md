# F_0017 Helper slash open/start/status

## 目标

验证真实飞书群内 helper slash contract：裸 `/helper` 返回 `/help` 或可用参数提示，具体 `/helper start` 打开启动卡，`/helper status` 返回当前状态。该用例只测 helper 控制面，不要求 helper 或 intern 执行业务任务。

- intern: `intern_ci_f_0017_worker_{run_id}`。

## 通过标准

- 裸 `/helper` 返回 usage/help 文案，不要求卡片。
- `/helper start` 打开可操作启动卡片。
- start callback 能启动 helper 或返回已 running。
- `/helper status` 回填当前 helper 状态。

## 失败分类

- 裸 `/helper` 返回 status/no-visible-helper 而不是 usage/help：产品 bug。
- `/helper start` 无卡片或卡片按钮缺失：产品 bug。
- start 返回成功但 helper registry 不存在：产品 bug。
- status 发到错误群：产品 bug。
