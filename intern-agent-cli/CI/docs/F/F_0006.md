# F_0006 Workspace Mode Switch Removed Contract

<!-- METADATA:TYPE=F,ID=F_0006,STATUS=ReviewDraft,OWNER=intern_ci_lead,SESSION=3 -->

## 目标

验证 workspace mode 在添加时固定，旧的 mode switch CLI / daemon API / relay API 均不可用。mode 迁移必须走独立 `workspace migrate-mode --repo-url <repo> --target <repo_dotdir|metadata_branch>` 流程。

## 资源

- Codeup nonprotected workspace: `ci_f_0006_codeup_{run_id}`。

## 剧本

| # | 类型 | 操作 | 注释 |
|---|------|------|------|
| 1 | action | `action_reset_case_namespace(case_id="F_0006")` | 清理本 case 的 workspace 记录。 |
| 2 | action | `action_cli_workspace_create(provider="codeup", mode="repo_dotdir", display_name=remote_display)` | 创建 remote workspace baseline。 |
| 3 | assert | `assert_removed_cli(command=["workspace","mode","set"])` | 旧 CLI 入口不可用。 |
| 4 | assert | `assert_removed_daemon_api(path="/api/workspaces/<id>/mode/set")` | daemon 旧 API 返回 not found。 |
| 5 | assert | `assert_removed_relay_api(path="/api/workspaces/<id>/mode/set")` | relay 旧 API 返回 not found。 |
| 6 | assert | `assert_workspace_mode(display_name=remote_display, mode="repo_dotdir")` | 失败调用不改变 workspace record。 |
| 7 | action | `action_cli_workspace_delete(display_name=remote_display, confirm=True)` | 全局删除只删除 relay workspace record。 |

## 通过标准

- 旧 mode switch surface 不存在。
- workspace record 的 mode 保持添加时的值。
- 删除 workspace 不删除远程代码仓库或仓库 metadata。
