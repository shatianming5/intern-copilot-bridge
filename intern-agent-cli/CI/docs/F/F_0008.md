# F_0008 Intern duplicate and invalid create rollback

<!-- METADATA:TYPE=F,ID=F_0008,STATUS=ReviewDraft,OWNER=intern_ci_lead,SESSION=2 -->

## 目标

验证 intern create 的错误路径：同 workspace 同名 intern 必须拒绝；坏 backend、缺 workspace、非法 intern 名称必须失败并回滚。

## 资源

- workspace: `ci_f_0008_workspace_{run_id}`。
- intern: `intern_ci_f_0008_worker_{run_id}`。

## 通过标准

- 所有错误都有结构化原因。
- 失败后不留下 intern metadata、session registry、Feishu group 或 relay registry。
- 已存在的合法 intern 不受错误请求影响。

## 失败分类

- duplicate 被允许：产品 bug。
- 失败后留下 group 或 metadata：产品 bug。
- case 删除了非本 namespace intern：CI 逻辑错误。
