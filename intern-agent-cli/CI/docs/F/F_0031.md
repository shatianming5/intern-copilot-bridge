# F_0031 TreeView top-level config status and refresh contract

<!-- METADATA:TYPE=F,ID=F_0031,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=15 -->

## 目标

验证 TreeView 顶层配置和插件状态入口：plugin meta item、format check toggle、language switch/reload prompt、`intern.refreshTree` 的错误展示和 last-good state，以及插件 status bar 的 daemon/relay/hooks 健康提示。Setup webview 相关命令不进入本 case。

## 资源

- debug machine: `debug-a` 或 `debug-b`。
- workspace: `ci_f_0031_workspace_{run_id}`。

## 通过标准

- 顶层 meta/config/statusbar 信息稳定，方便部署现场定位。
- format check、language switch 和 refreshTree 失败路径都有明确用户反馈。
- Setup webview 不被本 case 执行。

## 失败分类

- refreshTree 失败清空 last-good state：产品 bug。
- status bar 健康状态和 daemon/relay 实际状态不一致：产品 bug。
- 真实执行 setup webview 相关命令：CI 逻辑错误。
