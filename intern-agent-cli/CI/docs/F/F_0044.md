## 通过标准

- TreeView 对 Claude intern 使用 `intern-claude` context、terminal icon/type label、project-scoped command argument。
- 在线状态按 Claude live tmux/process 判定，不混用 Codex/Copilot 视觉或逻辑。
- create/restart Claude GUI command 是 CLI 的精确 wrapper。
- 不执行 Team/setup，不用真实 GUI 操作替代 CLI/source contract。
