  - `task_ci_f_0024_open_quickpick_{run_id}`。
  - `task_ci_f_0024_completed_{run_id}`。
  - `task_ci_f_0024_inprogress_{run_id}`。

## 通过标准

- Open 和 Completed task 可通过 GUI 删除，并调用 `internal task-delete`。
- 取消确认和 InProgress 拒绝不调用 CLI。
- 无 tree item 时 QuickPick 删除路径可用，无任务时提示清晰。
- 删除后 TreeView 刷新，report 记录 branch/commit evidence。

## 失败分类

- InProgress task 被删除：产品 bug，高优先级。
- GUI 删除缺少 project scope 导致跨 workspace 误删：产品 bug。
- QuickPick 列表不显示 task status/assignee：GUI contract bug。
- CI 清理非本 case task：CI 逻辑错误。
