# F_0029 Skill source TreeView projection and mutation

<!-- METADATA:TYPE=F,ID=F_0029,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=15 -->

## 目标

验证 workspace 下 Skill Sources TreeView：empty state、add source、package/skill 节点、打开 `SKILL.md`、update source、remove source cancel/confirm/cascade 和非法 source rollback。该用例只准备 Codex intern 能使用的 skill source，不覆盖 Copilot shared skills。

## 资源

- debug machine: `debug-a` 或 `debug-b`。
- workspace: `ci_f_0029_workspace_{run_id}`。
- skill package key: `ci_f_0029_pkg_{run_id}`。
- skill key: `ci_f_0029_pkg_{run_id}/skill_alpha`。
- local source dir: `/tmp/intern_agent_CI/F_0029/{run_id}/skill_source`。
- invalid source dir: `/tmp/intern_agent_CI/F_0029/{run_id}/invalid_source`。

## 通过标准

- add/update/remove source 的 GUI/CLI contract 一致。
- TreeView package、skill item、contextValue 和 `SKILL.md` open path 正确。
- 错误 source 不写入 registry，不自动启用 skill。

## 失败分类

- add source 自动启用 skill：产品 bug。
- remove 取消仍删除 source：产品 bug。
- 非法 source 留下半成品 registry：产品 bug。
