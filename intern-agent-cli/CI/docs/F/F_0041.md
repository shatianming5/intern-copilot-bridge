# F_0041 Real Feishu ingress slash and card callback contract

## 目标

全面覆盖真实 relay-local Feishu ingress 路径：slash message、main bot P2P、unmapped group、config/helper card callback 和 RBAC。该 case 必须走 relay 的 real message/card callback handler 或等价 CI-only source-driver；不能用 `/api/ci/feishu_message` 直接给 daemon 注入 `type=feishu_message` 来冒充 slash。

- intern: `intern_ci_f_0041_codex_{run_id}`。
- group/chat fixtures must be case-scoped.

## 通过标准

- 使用 real relay handler 或等价 CI-only source-driver，不走 synthetic daemon injection。
- `/config` Save/Cancel/RBAC、`/helper` usage/start/card、main bot P2P、unmapped group、unknown slash 都有可判定 evidence。
- 不 package/reset/deploy/bootstrap/restart relay。
