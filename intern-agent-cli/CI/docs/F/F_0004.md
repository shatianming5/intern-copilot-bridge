# F_0004 Local workspace add/remove

<!-- METADATA:TYPE=F,ID=F_0004,STATUS=ReviewDraft,OWNER=intern_ci_lead,SESSION=2 -->

## 目标

验证 debug 机器上的临时 local repo 可以通过 GUI/CLI 添加为 `local_only` workspace，并可删除。该用例必须使用 case 私有临时目录，不触碰当前开发机 workspace。

## 资源

- local repo root: `/root/axis_enterprise_ci/ci-artifacts/F_0004/local_repo_{run_id}`。
- workspace display: `ci_f_0004_local_{run_id}`。

## 剧本

| # | 类型 | 操作 | 注释 |
|---|------|------|------|
| 1 | action | `action_reset_case_namespace(case_id="F_0004")` | 初始化本地 workspace case 的 namespace，只清理本 case 生成的 registry 和临时路径。 |
| 2 | action | `action_create_temp_git_repo(path=local_repo, files={"README.md": "F0004 local workspace"})` | 创建受控临时 git repo，模拟用户把本机已有仓库添加为 workspace。 |
| 3 | action | `action_gui_add_workspace(provider="local", repo_path=local_repo, mode="local_only", display_name=display)` | 模拟 GUI 添加 local workspace，选择只能本地使用的 `local_only` 模式。 |
| 4 | assert | `assert_gui_command_cli_equivalent(gui_command="intern.addWorkspace", cli="internctl workspace create --provider local --mode local_only")` | 断言 GUI local 添加命令可由 CLI 完整表达。 |
| 5 | wait | `wait_for_workspace_registered(display_name=display)` | 等待 local workspace registry 写入完成。 |
| 6 | assert | `assert_workspace_record(provider="local", mode="local_only", repo_path=local_repo)` | 断言 registry 记录的是本地路径而不是远端 repo_url，且 mode 为 `local_only`。 |
| 7 | assert | `assert_no_remote_provider_credentials_used()` | 断言 local workspace 不会读取或发送 Codeup/GitHub 凭证，避免本地模式误触远端账号。 |
| 8 | action | `action_cli_workspace_delete(display_name=display, confirm=True)` | 通过 CLI 删除 local workspace registry 记录，验证 GUI 创建后 CLI 可管理。 |
| 9 | assert | `assert_workspace_removed(display_name=display)` | 断言 workspace 已从 registry 中删除。 |
| 10 | assert | `assert_temp_repo_preserved(path=local_repo)` | 断言删除 workspace 不会删除用户本地仓库内容，只移除插件配置。 |

## 通过标准

- local workspace 只引用 case 私有 repo path。
- 创建过程不读 Codeup/GitHub/Feishu 凭据。
- 删除 workspace 不删除 local repo 内容。

## 失败分类

- 写入当前开发机 workspace 或共享 registry：CI 逻辑错误。
- 删除 local repo 内容：产品 bug。
- GUI 无 CLI 等价入口：产品 bug。
