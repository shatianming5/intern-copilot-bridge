# F_0028 Codex Feishu group mode TreeView context commands

<!-- METADATA:TYPE=F,ID=F_0028,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=15 -->

## 目标

验证 VS Code TreeView 上 Codex intern 的 Feishu group mode context commands：`setTriggerModeAll`、`setTriggerModeAtOnly`、`setDetailModeFull`、`setDetailModeSummary` 与 `internctl group trigger-mode/detail-mode` 的 CLI contract 和 daemon/relay group config 一致。该用例不同于 slash `/config`，只覆盖 GUI 右键入口。

## 资源

- debug machine: `debug-a` 或 `debug-b`。
- workspace A: `ci_f_0028_workspace_a_{run_id}`。
- workspace B: `ci_f_0028_workspace_b_{run_id}`。
- intern: `intern_ci_f_0028_codex_{run_id}`。

## 通过标准

- 四个 GUI mode command 与 CLI 参数、daemon/relay group config 一致。
- 同名 intern 跨 workspace 不串改。
- 缺 group 失败有明确错误，且不留下半成品配置。

## 失败分类

- GUI command 缺 project 或改错 group：产品 bug。
- 缺 group 被当成成功：产品 bug。
- 使用 slash synthetic endpoint 替代 TreeView context command：CI 逻辑错误。
