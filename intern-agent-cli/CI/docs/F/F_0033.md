# F_0033 Codex no-prompt restart startup contract

<!-- METADATA:TYPE=F,ID=F_0033,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=65 -->

## 目标

验证 no-prompt Codex session 的 resume hint command 形状，以及 restart 后能重新进入 running。该 case 不发送自然语言 prompt；因此不要求存在 Codex session UUID，fresh restart 是正常结果。

用户发过消息后 `/exit` 提示命令可执行且 resume 同一个 Codex session id 属于 J 用例，因为它需要触发 agent turn。

## 资源

- namespace: `ci_f_0033`。
- workspace: `ci_f_0033_workspace_{run_id}`。
- intern: `intern_ci_f_0033_codex_{run_id}`。

## 通过标准

- Resume hint command 形状正确。
- restart 后 session/tmux/status running；fresh start 和 resume 都可接受。
- 不发送业务 prompt，不创建非 Codex intern，不依赖 Team/setup。

## 失败分类

- `product_bug_exit_resume_hint_command_invalid`
- `product_bug_restart_not_running`
