# F_0026 Codex active intern status bar and chat routing

<!-- METADATA:TYPE=F,ID=F_0026,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=15 -->

## 目标

验证 Codex intern 的 active selection、status bar、session resource map、daemon active route 更新和打开聊天入口。该用例只验证插件/CLI/session routing，不发送业务 prompt，不覆盖 Team 或非 Codex intern。

## 资源

- debug machine: `debug-a` 或 `debug-b`。
- workspace A: `ci_f_0026_workspace_a_{run_id}`。
- workspace B: `ci_f_0026_workspace_b_{run_id}`。
- duplicate intern: `intern_ci_f_0026_same_{run_id}`。
- session resources:
  - `vscode-chat-session://ci_f_0026/a/{run_id}`。
  - `vscode-chat-session://ci_f_0026/b/{run_id}`。

## 通过标准

- duplicate intern selection 使用 `name@project` disambiguation。
- active intern、status bar、daemon active route 和 session map 都使用 project scope。
- Open Chat 只建立聊天会话，不驱动 agent 做业务操作。

## 失败分类

- session map 以裸 intern name 覆盖同名 project：产品 bug。
- unknown session 不清 active intern：产品 bug。
- Open Chat 直接发送业务 prompt：产品 bug。
