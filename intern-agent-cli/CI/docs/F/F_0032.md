# F_0032 TreeView menu visibility and contextValue audit

<!-- METADATA:TYPE=F,ID=F_0032,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=15 -->

## 目标

验证 TreeView 的 `contextValue` 与 `package.json` menu `when` contract 在 Codex/workspace/task/skill 维度一致，避免用户看到错误菜单或看不到应有菜单。Team、Setup webview、非 Codex intern 和 Copilot shared skills 不纳入本 case。

## 资源

- debug machine: `debug-a` 或 `debug-b`。
- workspace: `ci_f_0032_workspace_{run_id}`。
- Codex intern: `intern_ci_f_0032_codex_{run_id}`。
- task: `task_ci_f_0032_open_{run_id}`。
- skill package: `ci_f_0032_pkg_{run_id}`。
- skill: `ci_f_0032_pkg_{run_id}/skill_alpha`。

## 通过标准

- Codex/workspace/task/skill 的 contextValue 与菜单 `when` contract 一致。
- 常用菜单可见且 command args 保持 project scope。
- Team、Setup、非 Codex intern、Copilot shared skills 不被执行。

## 失败分类

- contextValue 导致必要菜单不可见：产品 bug。
- command args 缺 project：产品 bug。
- 本 case 执行 out-of-scope Team/Setup/Copilot shared 命令：CI 逻辑错误。
