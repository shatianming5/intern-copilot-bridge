## 通过标准

- 三种 target resolution 可判定。
- 无机器/ambiguous 有结构化错误。
- Card 有 Cancel，Cancel 无副作用。
- owner/foreign operator 边界正确。
- Save 触发 policy sync 外壳，offline 有明确 warning。
