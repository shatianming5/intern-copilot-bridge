# F_0045 Claude skill farm and group parity contract

<!-- METADATA:TYPE=F,ID=F_0045,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=62 -->

## 目标

验证 Claude intern 的 skill farm 和 Feishu group 配置路径与 Codex parity：Claude 应使用 `.claude/skills`，repo/personal enable/disable/promote 不能写入 `.agents/skills`；group trigger/detail mode 的 CLI/API contract 对 Claude intern 同样成立。

本 case 不询问 Claude “能看到什么 skill”。那类需要 agent 自然语言判断的覆盖属于 J，除非后续能从文件/registry 直接脚本判定。

## 通过标准

- Claude skill sync 使用 `.claude/skills`，不写 Codex `.agents/skills`。
- repo/personal enable/disable/promote 与 TreeView projection、config 文件一致。
- Claude group trigger/detail mode 与 Codex 同等支持 project-scoped 配置。
- 不询问 Claude、不创建 Team、不执行 setup。
