# F_0019 Main bot readonly slash commands

<!-- METADATA:TYPE=F,ID=F_0019,STATUS=ReviewDraft,OWNER=intern_ci_lead,SESSION=2 -->

## 目标

验证主入口 bot 的只读 slash 命令：`/status`、`/list machines`、`/list workspaces`、`/list interns`、`/debug`。这些命令不能创建 workspace、intern、task 或触发 agent。

## 资源

- main bot test chat configured by CI Feishu app。
- deploy gate 已通过。

## 剧本

| # | 类型 | 操作 | 注释 |
|---|------|------|------|
| 1 | action | `action_reset_case_namespace(case_id="F_0019")` | 初始化 main bot readonly case 的 namespace，用于确认只读命令不会创建本 case 资源。 |
| 2 | action | `action_send_main_bot_slash(text="/status")` | 向 main bot 发送 `/status`，模拟主管查看系统整体状态。 |
| 3 | wait | `wait_for_main_bot_reply(command="/status")` | 等待 main bot 回复 status，避免异步消息延迟造成误判。 |
| 4 | assert | `assert_status_reply_contains(relay_running=True, machines_connected>=2)` | 断言 status 回复显示 relay running 且至少两台机器连接。 |
| 5 | action | `action_send_main_bot_slash(text="/list machines")` | 发送 `/list machines`，验证 main bot 机器列表查询。 |
| 6 | assert | `assert_list_reply_contains(items=["debug-a", "debug-b"])` | 断言机器列表包含固定部署的 debug-a 和 debug-b。 |
| 7 | action | `action_send_main_bot_slash(text="/list workspaces")` | 发送 `/list workspaces`，验证 workspace 列表查询。 |
| 8 | assert | `assert_list_reply_has_workspace_summary(redacted_secrets=True)` | 断言 workspace 回复是摘要信息，且不会泄露 token、key 或 repo 凭证。 |
| 9 | action | `action_send_main_bot_slash(text="/list interns")` | 发送 `/list interns`，验证 intern 列表查询。 |
| 10 | assert | `assert_list_reply_has_intern_summary(project_scoped=True)` | 断言 intern 列表按 project scope 展示，避免跨 project 混淆。 |
| 11 | action | `action_send_main_bot_slash(text="/debug")` | 发送 `/debug`，验证 main bot 的诊断信息入口。 |
| 12 | assert | `assert_debug_reply_contains(report_links=True, secrets_redacted=True)` | 断言 debug 回复包含可追踪 report 链接，同时所有 secret 脱敏。 |
| 13 | assert | `assert_no_resource_created_by_readonly_commands(case_id="F_0019")` | 断言 `/status`、`/list`、`/debug` 都是只读操作，没有创建 workspace、intern 或 group。 |

## 通过标准

- 只读命令都有可见回复。
- 回复包含机器/workspace/intern 基本信息，secret redacted。
- 无 workspace、intern、task、session 被创建。

## 失败分类

- 只读命令创建资源：产品 bug。
- secret/token 出现在回复：产品 bug。
- main bot 命令路由到 intern 群：产品 bug。
