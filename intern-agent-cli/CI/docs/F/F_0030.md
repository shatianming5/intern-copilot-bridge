# F_0030 Codex skill repo and personal enable contract

<!-- METADATA:TYPE=F,ID=F_0030,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=15 -->

## 目标

验证 Codex skill enable/disable 的 repo 和 personal scope：repo enable/disable、personal enable/disable、repo-enabled 时 personal reject、personal promote to repo、farm sync 和多 Codex intern 的影响边界。该用例不覆盖 Copilot shared skills，不创建 Claude intern。

## 资源

- debug machine: `debug-a` 或 `debug-b`。
- workspace: `ci_f_0030_workspace_{run_id}`。
- Codex interns:
  - `intern_ci_f_0030_a_{run_id}`。
  - `intern_ci_f_0030_b_{run_id}`。
- skill package key: `ci_f_0030_pkg_{run_id}`。
- skill key: `ci_f_0030_pkg_{run_id}/skill_alpha`。

## 通过标准

- repo/personal scope 的 GUI/CLI contract、registry 和 farm sync 一致。
- repo-enabled 时 personal enable 被拒绝或 no-op，promote to repo 会清理 personal holders。
- Codex intern 之间的 repo/personal 影响边界清楚。

## 失败分类

- personal enable 影响其他 Codex intern：产品 bug。
- repo disable 后 farm 残留：产品 bug。
- case 写入 Copilot shared state 或非 Codex fixture：CI 逻辑错误。
