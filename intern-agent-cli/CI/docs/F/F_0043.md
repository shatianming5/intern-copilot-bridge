# F_0043 Claude intern create/session lifecycle contract

<!-- METADATA:TYPE=F,ID=F_0043,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=65 -->

## 目标

验证 Claude intern 的创建、状态、Feishu group type、tmux session start/restart 基础生命周期。该 case 不要求 Claude 完成业务任务，不发送自然语言 prompt；只验证产品功能面是否能稳定启动、识别并通过 resume 路径重启 Claude session。

同时验证用户在 Claude 窗口中 `/exit` 后回到 tmux shell 时，产品打印的 `Resume this intern` 提示命令是可执行的；用户按提示在 tmux 中运行该命令后，必须 resume 之前同一个 Claude session uuid。

## 通过标准

- Claude intern 创建后在 list/status/session map/group registry 中均为 type=claude。
- Claude session start 后真实 Claude 进程 live，不只是 tmux shell。
- `/exit` 后 tmux pane 输出 `Resume this intern` 提示命令，且该命令可执行。
- 执行提示命令后 resume 同一个 Claude session uuid。
- restart 走 resume path，输出真实 uuid，并且连续 restart 的 uuid 一致。
- restart/resume 后最终飞书群仍为 Claude 绿灯。
- token/policy evidence 全部 redacted。
- 不发送业务 prompt，不创建 Team，不 package/reset/deploy/bootstrap/restart relay。
