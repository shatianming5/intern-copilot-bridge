# F_0025 Codex intern TreeView projection and project scope

<!-- METADATA:TYPE=F,ID=F_0025,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=15 -->

## 目标

验证 VS Code TreeView 对 Codex intern 的用户可见投影：`intern-codex` contextValue、rocket 图标、state/currentTask/PR tooltip、在线/离线分组、active focus 排序，以及跨 workspace 同名 intern 的 project scope 隔离。该用例不创建 Claude/Copilot intern，不覆盖 Team。

## 资源

- debug machine: `debug-a` 或 `debug-b`。
- workspace A: `ci_f_0025_workspace_a_{run_id}`。
- workspace B: `ci_f_0025_workspace_b_{run_id}`。
- shared intern name: `intern_ci_f_0025_same_{run_id}`。
- active intern: `intern_ci_f_0025_active_{run_id}`。
- task id: `task_ci_f_0025_focus_{run_id}`。

## 通过标准

- Codex intern 的 contextValue、图标、description、tooltip 与真实 status/session 状态一致。
- 同名 intern 在不同 workspace 下 project-scoped，不出现命令路由串线。
- 在线、离线和 active focus 排序符合 TreeView contract。

## 失败分类

- contextValue 或 command args 缺 project：产品 bug。
- 同名 intern 跨 workspace 串线：产品 bug。
- case 创建了非 Codex intern 或 Team state：CI 逻辑错误。
