# F_0054 Builtin default skill farm and protection contract

<!-- METADATA:TYPE=F,ID=F_0054,STATUS=ReviewDraft,OWNER=intern_ci_lead,SESSION=6 -->

## 目标

验证默认内置 skill 的文件/registry/UI 合约：`peer-send`、`goal-send`、
`feishu-messaging` 在 Codex/Claude skill sync 后自动进入对应 farm；它们不依
赖 repo/personal config，不在普通 Skill Sources/Repo/Personal TreeView 里当作
可编辑项展示；直接 add/update/remove/enable/disable 会被 protected-builtin
规则拒绝；farm 中陈旧或被篡改的 builtin 内容会在下一次 sync 被恢复。

本 case 不要求 agent 真实使用这些 skill。真实使用和 compaction 后可发现性
属于 `J_0012`。

## 通过标准

- Codex/Claude sync 自动启用三个 builtin skills。
- Builtin skills 不需要 repo/personal config。
- Builtin farm 内容来自当前 helper bundle，并可覆盖 stale/tampered 内容。
- Builtin skills 被 TreeView 普通 skill 管理面隐藏，并受 CLI protected
  mutation guard 保护。
- 同名 repo/personal skill 与 builtin 冲突时，builtin 胜出且给出清晰指导。
