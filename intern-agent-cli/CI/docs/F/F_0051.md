# F_0051 workspace local mode 切换拒绝合约

<!-- METADATA:TYPE=F,ID=F_0051,STATUS=ReviewDraft,OWNER=intern_ci_lead,SESSION=79 -->

## 目标

验证旧 workspace mode switch 入口不可用，尤其不能通过旧入口在远端 mode 与 `local_only` 之间切换。该用例不创建飞书群，不创建真实 intern，不启动 LLM。

## 阶段

F 类测试：CLI/daemon/GUI handler 合约。

本用例不属于 J，不留下飞书群现场，只通过 report 判断通过情况。

## 覆盖矩阵

1. Codeup nonprotected workspace: `repo_dotdir` -> `local_only`，拒绝。
2. Codeup nonprotected workspace: `metadata_branch` -> `local_only`，拒绝。
3. GitHub nonprotected workspace: `repo_dotdir` -> `local_only`，拒绝。
4. GitHub nonprotected workspace: `metadata_branch` -> `local_only`，拒绝。
5. Local workspace: `local_only` -> `repo_dotdir`，拒绝。
6. Local workspace: `local_only` -> `metadata_branch`，拒绝。

## 剧本

### Step 1：远端 mode 切到 local_only 全部拒绝

- action: `action_attempt_workspace_mode_change(workspace_id=codeup_dotdir_workspace, from_mode="repo_dotdir", to_mode="local_only")`
- assert: `assert_mode_change_rejected(result, from_mode="repo_dotdir", to_mode="local_only", reason_kind="local_mode_transition_forbidden")`
- assert: `assert_workspace_mode_unchanged(workspace_id=codeup_dotdir_workspace, metadata_mode="repo_dotdir")`
- action: `action_attempt_workspace_mode_change(workspace_id=codeup_meta_workspace, from_mode="metadata_branch", to_mode="local_only")`
- assert: `assert_mode_change_rejected(result, from_mode="metadata_branch", to_mode="local_only", reason_kind="local_mode_transition_forbidden")`
- assert: `assert_workspace_mode_unchanged(workspace_id=codeup_meta_workspace, metadata_mode="metadata_branch")`
- action: `action_attempt_workspace_mode_change(workspace_id=github_dotdir_workspace, from_mode="repo_dotdir", to_mode="local_only")`
- assert: `assert_mode_change_rejected(result, from_mode="repo_dotdir", to_mode="local_only", reason_kind="local_mode_transition_forbidden")`
- assert: `assert_workspace_mode_unchanged(workspace_id=github_dotdir_workspace, metadata_mode="repo_dotdir")`
- action: `action_attempt_workspace_mode_change(workspace_id=github_meta_workspace, from_mode="metadata_branch", to_mode="local_only")`
- assert: `assert_mode_change_rejected(result, from_mode="metadata_branch", to_mode="local_only", reason_kind="local_mode_transition_forbidden")`
- assert: `assert_workspace_mode_unchanged(workspace_id=github_meta_workspace, metadata_mode="metadata_branch")`

### Step 2：local_only 切到远端 mode 全部拒绝

- action: `action_attempt_workspace_mode_change(workspace_id=local_workspace, from_mode="local_only", to_mode="repo_dotdir")`
- assert: `assert_mode_change_rejected(result, from_mode="local_only", to_mode="repo_dotdir", reason_kind="local_mode_transition_forbidden")`
- assert: `assert_workspace_mode_unchanged(workspace_id=local_workspace, metadata_mode="local_only")`
- action: `action_attempt_workspace_mode_change(workspace_id=local_workspace, from_mode="local_only", to_mode="metadata_branch")`
- assert: `assert_mode_change_rejected(result, from_mode="local_only", to_mode="metadata_branch", reason_kind="local_mode_transition_forbidden")`
- assert: `assert_workspace_mode_unchanged(workspace_id=local_workspace, metadata_mode="local_only")`

## 通过标准

- 六个 mode change 请求都直接失败。
- 每次失败都有结构化错误，错误类别为 `local_mode_transition_forbidden`。
- 旧入口调用失败后 workspace mode、metadata root、provider、repo_url 均不变。
- report 记录每个请求的 command、returncode、payload、before mode 和 after mode。

## 失败分类

- 任何一个 local 相关转换成功：产品 bug。
- 转换失败但 workspace 已部分变更：产品 bug。
- 转换失败只返回 traceback、超时或不可解析输出：产品 bug 或 CLI 合约 bug，按证据分类。
- fixture 未能构造指定 workspace：CI fixture 问题。
