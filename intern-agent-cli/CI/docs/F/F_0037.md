# F_0037 Daemon reconnect registry and policy resync contract

<!-- METADATA:TYPE=F,ID=F_0037,STATUS=ApprovedForImplementation,OWNER=intern_ci_lead,SESSION=62 -->

## 目标

覆盖单机 daemon 断连/重连后的恢复能力：relay machine registry 恢复 online、workspace/chat lookup 仍可用、policy 重新拉取，无需 relay restart。

## 资源

- namespace: `ci_f_0037`。
- workspace: `ci_f_0037_workspace_{run_id}`。
- intern: `intern_ci_f_0037_codex_{run_id}`。
- debug machine: one existing debug machine。

## 通过标准

- 单机 daemon reconnect 后 relay registry online。
- workspace/chat lookup 恢复。
- policy 可重新拉取。
- 成功结束后不删群、不 stop session、不删 workspace，保留 case 现场证据。
- 不重启 relay，不做 package/reset/deploy/bootstrap。
