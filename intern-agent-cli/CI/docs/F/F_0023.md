## 通过标准

- Task TreeView 分组、计数、默认展开/收起符合 GUI contract。
- README 存在才打开，缺失时不崩溃。
- Tooltip 中 status、assignee、原始 PR 字段稳定；InProgress description 对 PR 做 `PR#N` 格式化。
- METADATA 第 3 行规则生效，畸形 task 不被误展示为正常任务；当前 CLI 若接受第 4 行 METADATA，按产品/parser contract bug 报告。

## 失败分类

- TreeView 分组/tooltip 与 metadata 不一致：产品 bug。
- 缺 README 导致 GUI 抛异常：产品 bug。
- 畸形 metadata 被当作正常 task：产品/parser contract bug。
- 断言 Open unassigned task 同时带 PR：剧本或 CI harness 错误，因为当前真实 CLI contract 无法表达该状态。
- case seed 的 task 污染其他 workspace：CI 逻辑错误。
