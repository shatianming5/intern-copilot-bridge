# Screenshot font resources

`NotoSansSC-Regular.otf` is vendored from the Noto CJK project:

https://github.com/notofonts/noto-cjk/blob/main/Sans/SubsetOTF/SC/NotoSansSC-Regular.otf

It is used only as a CJK glyph fallback for `/screenshot` PNG rendering. The
renderer still positions text on a monospace terminal grid so ASCII output and
line numbers remain aligned.
